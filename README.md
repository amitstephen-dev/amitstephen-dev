## 🎯 Featured Projects

### GoRelay

A durable background job library for Go that turns functions into crash-resistant, retryable tasks.

**Key Features**

* SQLite, PostgreSQL, and Redis backends
* Automatic retries with exponential backoff
* Scheduled jobs
* Priority queues
* Web dashboard
* Zero-infrastructure setup

```bash
go get github.com/amitstephen-dev/gorelay
```

### GoHeapGuard

A lightweight memory monitoring and observability tool for Go applications focused on understanding heap behavior, memory usage patterns, and runtime performance.
